# Cloning

 ```
 git clone --recursive git@github.com:mlanca-c/Minitalk.git
 cd Minitalk
 make all
 ```
 
 # Compiling
 
 ```
 $ ./server 
 PID: 25928
 Hello World, this is Minitalk!
 
 ```

 ```
 $ ./client 25928 "Hello World, this is Minitalk!"
 ```
